= Cloud Volumes ONTAP 배포에 지원되는 ONTAP 버전
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
NetApp Console 사용하면 Cloud Volumes ONTAP 시스템을 추가할 때 여러 가지 ONTAP 버전 중에서 선택할 수 있습니다.

여기에 나열된 버전 이외의 Cloud Volumes ONTAP 버전은 신규 배포에 사용할 수 없습니다. 릴리스에 표시된 패치 버전 또는 일반(General Availability) 버전은 배포에 사용할 수 있는 기본 버전을 나타냅니다. 사용 가능한 패치에 대한 자세한 내용은 각 릴리스의 https://docs.netapp.com/us-en/cloud-volumes-ontap-relnotes/reference-new.html["버전별 릴리스 노트"^]를 참조하십시오.

업그레이드에 대한 자세한 내용은 link:task-updating-ontap-cloud.html#supported-upgrade-paths["지원되는 업그레이드 경로"]을(를) 참조하십시오.
