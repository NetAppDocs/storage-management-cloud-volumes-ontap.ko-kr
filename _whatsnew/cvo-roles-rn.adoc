=== 탭별로 수행할 수 없는 활동

다음 표에는 탭별로 제한된 활동이 요약되어 있습니다.

[cols="1,4"]
|===
| 탭 | 수행할 수 없는 활동 


| 개요  a| 
* 볼륨 추가
* 애그리게이트 추가
* System Manager를 엽니다
* 켜기/끄기 버튼 사용
* 암호 설정
* ONTAP 버전 업데이트
* 작업 공간에서 시스템을 제거합니다
* 시스템을 삭제합니다
* *지금 업그레이드* 사용
* 전체 기능 연필 메뉴에서 *시스템 태그* 편집
* 모든 기능 연필 메뉴에서 *예정된 다운타임* 편집
* 전체 기능 연필 메뉴에서 *Storage classes*를 편집하세요.
* 모든 기능 연필 메뉴에서 *Instance type*을 편집하세요.
* 모든 기능 연필 메뉴에서 *충전 방식* 편집
* 모든 기능 연필 메뉴에서 *라이선스 유형* 편집
* 모든 기능 연필 메뉴에서 *쓰기 속도* 편집
* 전체 기능 연필 메뉴에서 *랜섬웨어 보호*를 편집하세요.
* 전체 기능 연필 메뉴에서 *Support registration*을 편집합니다
* 모든 기능 연필 메뉴에서 *WORM*을 편집합니다
* 모든 기능 연필 메뉴에서 *Data tiering*을 편집하세요.
* 모든 기능 연필 메뉴에서 *CIFS 설정*을 편집합니다
* 모든 기능 연필 메뉴에서 *경로 테이블* 편집




| 볼륨  a| 
이 탭에서는 볼륨 업데이트가 허용되지 않으며 볼륨을 추가할 수 없습니다.

* 볼륨 복제
* 볼륨 설정 편집
* 볼륨 삭제
* 스냅샷 복사본 생성
* 스냅샷 복사본에서 복원
* 디스크 유형 변경(고급 옵션)
* 계층화 정책 변경(고급 옵션)




| 애그리게이트  a| 
* 애그리게이트 추가
* 볼륨 추가
* 용량 증대
* 애그리게이트를 삭제합니다




| 복제  a| 
* 복제 중단
* 복제 재동기화
* 역방향 재동기화 복제
* 일정 편집
* 최대 전송 속도 편집
* 복제 업데이트
* 복제 삭제




| 메인 캔버스  a| 
* 시스템 추가
* 시스템 검색(모든 운영자용)
* *Copy and sync*에 드래그 앤 드롭 사용
* *Volume caching*에 드래그 앤 드롭 사용
* *복제*에는 드래그 앤 드롭을 사용하십시오
* 볼륨 추가
* *최적화된 비용 및 성능* 액세스
* System Manager를 엽니다
* 필요한 CVO 권한을 사용할 수 있는 경우에만 Cloud Volumes ONTAP에 *복제*를 사용하십시오(FSx for ONTAP 또는 온프레미스 시스템에는 적용되지 않음).
* 필요한 CVO 권한을 사용할 수 있는 경우에만 Cloud Volumes ONTAP에 대해 *볼륨 캐싱*을 사용하십시오(FSx for ONTAP 또는 온프레미스 시스템에는 적용되지 않음).
* 시스템을 입력합니다
* AVS 관리 버튼을 사용합니다




| 역할 없이  a| 
역할이 지정되지 않은 경우 모든 작업이 비활성화됩니다.

* 작업 환경에 액세스합니다
* 복제 액세스
* Cloud Volumes ONTAP 생성 마법사를 사용하십시오
* 복제 생성 마법사를 사용하십시오
* Cloud Volumes ONTAP 살펴보기


|===
